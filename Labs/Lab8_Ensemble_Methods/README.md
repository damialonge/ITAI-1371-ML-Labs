# Lab 8 – Ensemble Methods

## Overview
This lab covered ensemble learning — a powerful approach that combines multiple models to produce better predictions than any single model alone.

## What I Learned
- The concept of **ensemble learning** and why it works
- **Bagging** (Bootstrap Aggregating) and how Random Forest uses it
- **Boosting** (AdaBoost, Gradient Boosting) and how it corrects errors iteratively
- How **Random Forest** builds many decision trees and averages their results
- Feature importance — how ensemble models reveal which features matter most

## Challenges Faced
Understanding the difference between bagging and boosting, and when to use one over the other, required careful comparison. Gradient Boosting's sequential nature was particularly interesting to wrap my head around.
