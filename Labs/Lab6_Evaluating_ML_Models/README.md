# Lab 6 – Evaluating Machine Learning Models

## Overview
This lab focused on how to properly evaluate whether a machine learning model is actually performing well — going beyond simple accuracy.

## What I Learned
- Key evaluation metrics: **accuracy**, **precision**, **recall**, **F1-score**
- How to read and interpret a **confusion matrix**
- The difference between training accuracy and test accuracy
- **Cross-validation** as a more reliable evaluation method
- Why accuracy alone can be misleading (especially with imbalanced datasets)

## Challenges Faced
Understanding the trade-off between precision and recall — and knowing which one to prioritize depending on the use case — was the most thought-provoking part of this lab.

## Tools Used
- Python
- Google Colab
- scikit-learn (classification_report, confusion_matrix)
