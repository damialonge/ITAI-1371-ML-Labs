# Lab 9 – Unsupervised Learning

## Overview
This lab introduced unsupervised learning — finding patterns in data without labeled outputs — including clustering and dimensionality reduction.

## What I Learned
- The difference between supervised and unsupervised learning problems
- **K-Means Clustering**: how it groups data into k clusters
- How to choose the right number of clusters using the **Elbow Method**
- **Principal Component Analysis (PCA)** for dimensionality reduction
- How unsupervised learning can be used for anomaly detection (very relevant to cybersecurity!)

## Challenges Faced
Since there are no labels to validate against, evaluating unsupervised model performance felt less straightforward. Understanding how PCA reduces dimensions while retaining variance took some re-reading.

## Tools Used
- Python
- Google Colab
- scikit-learn (KMeans, PCA), matplotlib
