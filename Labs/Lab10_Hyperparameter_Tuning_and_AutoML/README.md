# Lab 10 – Hyperparameter Tuning and AutoML

## Overview
This lab explored how to systematically improve model performance by tuning hyperparameters, and introduced AutoML tools that automate parts of the ML pipeline.

## What I Learned
- The difference between **parameters** (learned by the model) and **hyperparameters** (set before training)
- **Grid Search** and **Random Search** for finding optimal hyperparameters
- How **cross-validation** is used during hyperparameter tuning to prevent overfitting
- An introduction to **AutoML** — tools that automate model selection and tuning
- How to use `GridSearchCV` in scikit-learn

## Challenges Faced
Grid search can be computationally expensive — understanding how to narrow the search space efficiently (rather than trying every combination) was an important practical lesson.

## Tools Used
- Python
- Google Colab
- scikit-learn (GridSearchCV, RandomizedSearchCV)
