# Lab 4 – Data Preparation and Feature Engineering

## Overview
This lab focused on one of the most critical steps in any ML project — preparing raw data and engineering meaningful features for modeling.

## What I Learned
- How to handle **missing values** (imputation strategies)
- **Encoding categorical variables** (label encoding, one-hot encoding)
- **Feature scaling** (normalization vs. standardization)
- How to create new features from existing ones to improve model performance
- The impact that data quality has on model accuracy

## Challenges Faced
Deciding which imputation strategy to use and understanding when to scale features versus when it wasn't necessary were tricky concepts to work through.
