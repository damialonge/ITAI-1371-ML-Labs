# Lab 3 – ML Workflow and Types of Learning

## Overview
This lab covered the end-to-end machine learning workflow and explored the three main types of machine learning: supervised, unsupervised, and reinforcement learning.

## What I Learned
- The standard ML workflow: problem definition → data collection → preprocessing → modeling → evaluation → deployment
- The difference between **supervised**, **unsupervised**, and **reinforcement learning**
- When to apply each type of learning based on the problem
- How labels and features play a role in supervised learning

## Challenges Faced
